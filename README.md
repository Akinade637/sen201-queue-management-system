# sen201-queue-management-system
The Queue Management System is a simple application that manages people waiting for service using the First-Come-First-Served (FCFS) approach. It allows users to add individuals to a queue, serve the next person, and view the current queue. The system is implemented in Python and demonstrates the key stages of the Software Development Life Cycle (SDLC).
